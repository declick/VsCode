Quizz-App

QuizZ-App is an interactive game that I created. The goal of the game is to reconstruct words from shuffled letters. To achieve this, I utilized various elements of the Document Object Model (DOM) to manage the user interface

The game features several functionalities, including selecting word categories, displaying shuffled words, letter manipulation (filling in empty slots), and checking player responses. I also integrated visual elements such as stylized alerts, progress indicators, and a life counter.

To make the user experience more dynamic, I included features such as the use of bonus stars to assist players, score saving and restoration, and the ability to reset the game.

Overall, QuizZ-App aims to provide an entertaining and engaging gaming experience with a responsive interface and interactive features.
