Projet Final

The project I've shared on GitHub is a web application that I developed using a combination of Front-End and Back-End technologies. For design, I opted to use Figma, an online tool, instead of Photoshop, to create wireframes and mockups.

At the application level, I implemented access levels to manage different permissions within the application. For the Front-End, I used React, primarily for the view in the MVC model. State management in React components is handled by the Context API as a dependency injection mechanism.

I implemented CRUD operations (Create, Read, Update, Delete) for data persistence, using Axios to send HTTP requests from the React client to the Back-End server. Navigation between application pages is managed by React Router.

Use of React for the Front-End. React is mainly used for the view (V) in the Model-View-Controller (MVC) model. The Context API is not used as a traditional state management system but rather as a dependency injection mechanism to manage state in a React component. Implementation of CRUD operations (Create, Read, Update, Delete) for data persistence. Axios is used to send HTTP requests from the React client to the Back-End server. React Router is used for navigation between different pages of the application.

Regarding the database, I defined the necessary structure to store required information. Concerning security, I established a registration and login system, likely using JSON Web Tokens (JWT) for authentication and authorization.

In summary, the project is a comprehensive web application, with particular attention to design, security, and data management through CRUD operations.
